# ex17-lines-pipeline

## Enunciado

Aplica un pipeline de limpieza sobre una lista de líneas de texto:
1. `trim`
2. eliminar vacías/blancas
3. `toLowerCase`

## Ejemplos
- Entrada: `["  Hola ", "", " Mundo  "]` → Salida: `["hola","mundo"]`
- Entrada: `["   ", "\t", "X"]` → Salida: `["x"]`

## Qué practicar (FP)
- Pipeline declarativo con streams.
