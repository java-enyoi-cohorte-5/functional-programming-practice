# ex07-flatten-lists

## Enunciado

Recibe una lista de listas de enteros y retorna una sola lista con todos los elementos en el mismo orden de aparición.

## Ejemplos
- Entrada: `[[1,2], [], [3]]` → Salida: `[1,2,3]`
- Entrada: `[]` → Salida: `[]`

## Qué practicar (FP)
- `flatMap` para aplanar estructuras anidadas.
