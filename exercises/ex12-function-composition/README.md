# ex12-function-composition

## Enunciado

Recibe dos funciones `f` y `g` y un valor inicial `x`.
Retorna el resultado de componer como `f(g(x))`.

## Ejemplos
- `f(x)=x+10`, `g(x)=x*2`, `x=5` → `f(g(5)) = 20`
- `f(x)=x`, `g(x)=x+3`, `x=7` → `10`

## Qué practicar (FP)
- Composición con `Function.compose` o `andThen`.
