# ex16-record-mapping

## Enunciado

Recibe una lista de `User(name, email)` y retorna una lista con los emails en minúscula.

## Reglas
- Conservar el orden de los usuarios.

## Ejemplos
- Entrada: `[("Ana", "ANA@MAIL.COM")]` → Salida: `["ana@mail.com"]`
- Entrada: `[("B","B@X.COM"), ("A","A@X.COM")]` → Salida: `["b@x.com","a@x.com"]`

## Qué practicar (FP)
- records + `map` + normalización de strings.
