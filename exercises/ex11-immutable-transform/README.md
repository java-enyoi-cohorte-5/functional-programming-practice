# ex11-immutable-transform

## Enunciado

Aplica un prefijo a cada string de entrada y retorna una nueva lista.

## Reglas
- No mutar la colección original.
- El resultado debe ser no modificable.

## Ejemplos
- Entrada: `["a","b"], "pre-"` → Salida: `["pre-a","pre-b"]`
- La lista original debe permanecer igual.

## Qué practicar (FP)
- `map` + recolección inmutable.
