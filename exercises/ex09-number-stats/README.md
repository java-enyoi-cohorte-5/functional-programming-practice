# ex09-number-stats

## Enunciado

Calcula estadísticas de una lista de enteros: cantidad, suma, mínimo, máximo y promedio.

## Reglas
- Debes retornar un `IntSummaryStatistics`.

## Ejemplos
- Entrada: `[2,4,6]` → count=`3`, sum=`12`, min=`2`, max=`6`, avg=`4.0`
- Entrada: `[-10,-5]` → min=`-10`, max=`-5`

## Qué practicar (FP)
- `mapToInt` + `summaryStatistics`.
