# ex06-stable-deduplicate

## Enunciado

Elimina elementos duplicados de una lista de strings, conservando la **primera aparición** de cada valor.

## Reglas
- Preservar el orden relativo original.
- No mutar la entrada.

## Ejemplos
- Entrada: `["a", "b", "a", "c", "b"]` → Salida: `["a", "b", "c"]`
- Entrada: `["x", "x", "x"]` → Salida: `["x"]`

## Qué practicar (FP)
- `distinct` y encounter order.
