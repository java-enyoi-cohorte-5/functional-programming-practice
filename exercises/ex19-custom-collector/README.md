# ex19-custom-collector

## Enunciado

Construye un collector personalizado para unir strings:
- convertir cada valor a mayúscula
- separarlos con `|`

## Reglas
- Para lista vacía, retorna string vacío.

## Ejemplos
- Entrada: `["a","b","c"]` → Salida: `"A|B|C"`
- Entrada: `[]` → Salida: `""`

## Qué practicar (FP)
- `Collector.of`.
