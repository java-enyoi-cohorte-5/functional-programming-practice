# ex14-group-by-initial

## Enunciado

Agrupa una lista de palabras por su letra inicial en minúscula.

## Reglas
- La clave del mapa es el primer carácter en minúscula.
- Mantener el orden relativo de palabras dentro de cada grupo.

## Ejemplos
- Entrada: `["Apple","apricot","Banana"]` → Salida: `{a:["Apple","apricot"], b:["Banana"]}`
- Entrada: `["Zoo"]` → Salida: `{z:["Zoo"]}`

## Qué practicar (FP)
- `groupingBy` con transformación de clave.
