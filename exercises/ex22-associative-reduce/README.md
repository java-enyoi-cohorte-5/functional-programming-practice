# ex22-associative-reduce

## Enunciado

Calcula el producto de una lista de enteros mediante `reduce`.

## Reglas
- Identidad correcta: `1`.
- Si la lista está vacía, retorna `1`.

## Ejemplos
- Entrada: `[2,3,4]` → Salida: `24`
- Entrada: `[]` → Salida: `1`
- Entrada: `[5,0,3]` → Salida: `0`

## Qué practicar (FP)
- `reduce(identity, accumulator)` y asociatividad.
