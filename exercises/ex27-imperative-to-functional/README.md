# ex27-imperative-to-functional

## Enunciado

Refactoriza la lógica para calcular la suma de longitudes de strings:
- ignora strings vacíos o en blanco
- transforma a mayúscula
- suma las longitudes resultantes

## Ejemplos
- Entrada: `["a", "", "bc"]` → Salida: `3`
- Entrada: `[" ", "x"]` → Salida: `1`

## Qué practicar (FP)
- `filter` + `map` + `mapToInt` + `sum`.
